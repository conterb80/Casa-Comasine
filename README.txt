CASA COMASINE PWA

Questa cartella è pronta per essere pubblicata su un hosting statico.
Quando la pubblichi online, il browser Android potrà proporre "Aggiungi a schermata Home"
come vera app web installabile.

Per usare le tue foto come sfondo:
- sostituisci questi file mantenendo lo stesso nome:
  assets/bg-placeholder-1.jpg
  assets/bg-placeholder-2.jpg
  assets/bg-placeholder-3.jpg

Dove metterle:
- bg-placeholder-1.jpg = sfondo hero principale
- bg-placeholder-2.jpg = primo riquadro foto
- bg-placeholder-3.jpg = secondo riquadro foto
